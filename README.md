## 📖 Sobre

A **Biblioteca de Filmes** é uma Single Page Application (SPA) desenvolvida utilizando **HTML5**, **CSS3**, **JavaScript** e **Supabase**, organizada seguindo o padrão **MVC (Model-View-Controller)**.

---

## ✨ Funcionalidades

### Área Pública

- Página Inicial
- Catálogo de Filmes
- Pesquisa por título
- Página Sobre
- Página de Contato

### Área Administrativa

- Login
- Cadastro de filmes
- Edição de filmes
- Exclusão de filmes
- Listagem de filmes

---

## 🏗 Arquitetura MVC

```text
js/
├── config/
├── controllers/
├── models/
├── views/
└── app.js
```

- **Model:** Comunicação com o Supabase.
- **View:** Interface da aplicação.
- **Controller:** Regras de negócio.
